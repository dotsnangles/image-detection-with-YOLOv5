# image-detection-with-YOLOv5

### 구성원  

박정현, 김수현, 장재성, 소국희

### 프로젝트명  

생활 폐기물 객체 인식

### 프로젝트 목표

크롤링한 데이터와 YOLOv5를 활용한 객체 인식 모델 생성

### 역할  

크롤링, 데이터 전처리 및 어그멘테이션, 어노테이션 파일 생성, 모델 훈련, 추가 데이터 전처리 및 후속 훈련
